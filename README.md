# Instagram Clone

This is a responsive Instagram Clone project developed using HTML, CSS, and JavaScript. The main goal of this project is to recreate the look and feel of Instagram while practicing front-end web development concepts.

## Features
- Responsive user interface
- Instagram-like home feed
- Stories section
- Navigation bar
- User profile layout
- Like, comment, and share icons
- Clean and modern design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was built as a learning exercise to improve my skills in web development, responsive design, and Git & GitHub. It demonstrates how popular social media interfaces can be recreated using front-end technologies.
